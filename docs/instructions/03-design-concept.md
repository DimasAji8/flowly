# 03 — Design Concept

## Visual Style

Gunakan pendekatan desain:

- **Minimal**
- **Soft**
- **Modern**
- **Elegant**
- **Calm financial journal**

## Prioritas Desain

Yang ditekankan:

1. **Spacing** — beri ruang nafas antar elemen
2. **Typography** — hierarki jelas, font readable
3. **Layout** — struktur tenang & terorganisir

Bukan dekorasi berlebihan.

## Mood

Bayangkan aplikasi seperti **journal keuangan personal** — tenang, rapi, dan tidak menuntut perhatian berlebih. User membuka app, mencatat, lalu pergi tanpa gangguan visual.

## Referensi Mood (yang DIINGINKAN)

- Notion-like calm interface
- Apple Notes simplicity
- Things 3 typography & spacing
- Linear minimal aesthetic

## Anti-Reference (yang DIHINDARI)

- Crypto trading dashboard (Binance, dll)
- Enterprise accounting (SAP, Quickbooks)
- AI app dengan gradient ungu-biru
- Fintech yang full glassmorphism
