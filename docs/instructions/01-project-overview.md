# 01 — Project Overview

## Nama Produk

**Flowly** (placeholder name) — aplikasi mobile-first untuk pencatatan cashflow pribadi & multi-user.

## Tujuan Produk

Aplikasi yang memudahkan user mencatat dan memantau arus kas harian dengan pengalaman yang **tenang, cepat, dan personal** — bukan terasa seperti software akuntansi perusahaan.

## Fokus Utama

- Minimalis
- Clean
- Nyaman digunakan setiap hari
- Mobile-first
- Cepat saat input transaksi
- Visual tenang dan modern
- Tidak terasa seperti aplikasi accounting perusahaan

## Use Cases

Aplikasi digunakan untuk:

- Mencatat pemasukan dan pengeluaran
- Melihat histori transaksi
- Melihat cashflow berdasarkan kalender
- Mengelola kategori
- Recurring transaction
- Multi-user sederhana (workspace based)

## Filosofi Desain

> **calm UI + whitespace + typography + spacing**

Aplikasi ini **BUKAN**:

- Dashboard crypto
- UI penuh icon
- AI gradient style
- Glassmorphism berlebihan
- ERP / accounting enterprise

## Final Goal

Aplikasi harus terasa:

- Ringan
- Tenang
- Cepat
- Modern
- Personal
