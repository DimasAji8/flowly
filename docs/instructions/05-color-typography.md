# 05 — Color Palette & Typography

## Color Palette

Gunakan warna **soft matte**, bukan putih terang. Semua warna mengikuti gaya tenang.

| Token | Hex | Penggunaan |
|-------|-----|------------|
| Background | `#F3F4F6` | Background utama page |
| Surface | `#E5E7EB` | Surface elemen besar (header, section) |
| Card | `#ECEFF3` | Background card |
| Primary Text | `#111827` | Heading & body utama |
| Secondary Text | `#4B5563` | Caption, helper text |
| Accent | `#2563EB` | CTA, link, focus state |
| Success | `#15803D` | Income, positive state |
| Danger | `#B91C1C` | Expense, error, destructive |
| Border | `#D1D5DB` | Border tipis & separator |

### Rules Warna

- Tidak boleh pakai `#FFFFFF` murni untuk background.
- Income selalu hijau (`Success`).
- Expense selalu merah (`Danger`).
- Accent hanya dipakai untuk action utama (jangan terlalu sering).

## Typography

### Font

- **Inter** (primary font)

### Rules

- Heading **tegas** (font-weight 600–700)
- Body **clean** (font-weight 400–500)
- Line height **nyaman** (umumnya 1.5–1.6 untuk body)
- Tidak terlalu padat — beri ruang vertikal

### Suggested Scale

| Level | Size | Weight | Use Case |
|-------|------|--------|----------|
| H1 | 28–32px | 700 | Page title |
| H2 | 22–24px | 600 | Section title |
| H3 | 18–20px | 600 | Card title |
| Body | 14–16px | 400 | Konten utama |
| Caption | 12–13px | 400 | Helper, timestamp |

## Dark Mode

Tetap gunakan filosofi **matte color & soft contrast**.

- Bukan pure black (`#000000`).
- Background gelap suggested: matte dark gray (mis. `#111827` atau `#0F172A`).
- Pastikan kontras text tetap nyaman dibaca tapi tidak menyilaukan.
